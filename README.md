# GLPI-11-NGINX-MySQL-Zabbix7-Integration

## 📋 Visão Geral

Este projeto documenta a instalação e configuração integrada de:
- GLPI (Helpdesk)
- Zabbix 7.2 (Monitoramento)
- Grafana (Dashboards)
- NGINX + MySQL
